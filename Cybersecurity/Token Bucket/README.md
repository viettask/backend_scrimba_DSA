## How to Play

### Running the Game

```bash
node token_bucket_game.js
```

Or with npm:

```bash
npm start
```

### Game Rules

- **Bucket Capacity**: maximum number of tokens 
- **Refill Rate**: number of tokens that refill per second
- **Time Step**: Each round represents 1 second passing
- **Starting Tokens**: Bucket starts full

### Commands

- Enter a **number** to make that many requests
- Type `reset` to restart the game
- Type `quit` or `exit` to end the game
