2. chúng ta có 3 môi trường
    Dev 
    - phát triển dành cho developer 
    - mọi ng cùng đóng góp ý kiển

    Stagging 
    - kiểm thử, thuyết trình và kiểm tra nội bộ trong nhóm
    - khi chức năng đã xây dựng thành công thì tester sẽ test chức năng đó

    Production
    - chạy thật
    - môi trường cho mọi ng đều biết
    - nội dung ở phiên bản này đã sẵn sàng cho ng dùng cuối cùng
